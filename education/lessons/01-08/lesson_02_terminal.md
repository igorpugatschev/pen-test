# Занятие 2: Терминал Linux — базовые команды

## Теория

Терминал (shell) — основной интерфейс взаимодействия с Linux. Bash (Bourne Again Shell) — стандартная оболочка в большинстве дистрибутивов.

### Основные команды

| Команда | Назначение | Пример |
|---------|-----------|--------|
| `ls` | Список файлов | `ls -la` — подробный список |
| `cd` | Смена директории | `cd /etc` |
| `cp` | Копирование | `cp file1 file2` |
| `mv` | Перемещение/переименование | `mv old.txt new.txt` |
| `rm` | Удаление | `rm -rf dir/` (осторожно!) |
| `cat` | Вывод содержимого файла | `cat file.txt` |
| `less` | Постраничный просмотр | `less log.txt` |
| `man` | Справка по командам | `man ls` |

### Полезные ключи

- `ls -l` — подробный вывод (права, владелец, размер, дата)
- `ls -a` — показать скрытые файлы (начинаются с `.`)
- `ls -h` — человекочитаемый размер (K, M, G)
- `cp -r` — рекурсивное копирование директорий
- `rm -i` — интерактивное подтверждение удаления
- `cat -n` — вывод с номерами строк

### Специальные символы

- `~` — домашняя директория
- `.` — текущая директория
- `..` — родительская директория
- `*` — любые символы (glob)
- `?` — один любой символ
- `|` — пайп (передача вывода одной команды в другую)
- `>` — перенаправление вывода (перезапись)
- `>>` — перенаправление вывода (добавление)

## Практическое занятие

### Шаг 1: Работа с ls и навигация

```bash
# Подробный список файлов в домашней директории
ls -la ~

# Список файлов с человекочитаемым размером
ls -lah /var/log/

# Показать только скрытые файлы
ls -d .*

# Перейти в /etc и посмотреть файлы, отсортированные по времени
cd /etc
ls -lt
```

### Шаг 2: Копирование и перемещение

```bash
# Создать тестовую директорию
mkdir ~/terminal_practice
cd ~/terminal_practice

# Создать несколько файлов
touch file1.txt file2.txt file3.txt

# Копировать файл
cp file1.txt file1_copy.txt

# Копировать директорию (рекурсивно)
mkdir dir1
cp -r dir1 dir1_copy

# Переименовать файл
mv file2.txt renamed_file.txt

# Переместить файл в другую директорию
mv renamed_file.txt dir1/
```

### Шаг 3: Просмотр содержимого файлов

```bash
# Создать файл с содержимым
echo "Строка 1" > test.txt
echo "Строка 2" >> test.txt
echo "Строка 3" >> test.txt

# Просмотр через cat
cat test.txt

# Просмотр с номерами строк
cat -n test.txt

# Постраничный просмотр (q для выхода)
less /etc/passwd

# Просмотр первых/последних строк
head -5 /etc/passwd
tail -5 /etc/passwd
tail -f /var/log/syslog  # следить за логом в реальном времени
```

### Шаг 4: Удаление и перенаправление

```bash
# Удалить файл
rm file3.txt

# Удалить директорию с содержимым
rm -rf dir1_copy

# Перенаправление вывода
ls -la > file_list.txt
cat file_list.txt

# Пайп — передать вывод ls в grep
ls -la /etc | grep passwd

# Цепочка команд
cat /etc/passwd | head -10 | tail -5
```

### Шаг 5: Работа со справкой (man)

```bash
# Открыть справку по ls
man ls
# Пробел — следующая страница, q — выход, / — поиск

# Краткая справка (если доступна)
ls --help

# Поиск по справке (встроенная команда bash)
help cd
```

## Задачи для самостоятельного выполнения

1. Создайте директорию `~/practice` и внутри неё создайте структуру: 3 файла `report1.txt`, `report2.txt`, `report3.txt` и папку `archive/`. Скопируйте все `.txt` файлы в `archive/`.

2. Найдите все файлы в `/etc`, содержащие в названии "conf", и запишите их список в файл `~/conf_files.txt`. Используйте `ls` и пайп с `grep`.

3. Используя `cat` и перенаправление `>`, создайте файл `contacts.txt` с тремя записями (имя, email) на разных строках. Добавьте четвертую запись с помощью `>>`.

4. Изучите справку (`man`) по командам `cp` и `mv`. Найдите ключ, который позволяет сохранять атрибуты файла (владелец, группа, временные метки) при копировании. Запишите этот ключ.

5. Выведите содержимое `/etc/passwd` постранично через `less`. Найдите в файле всех пользователей, у которых оболочка — `/bin/bash` (используйте поиск внутри `less` по `/bash`).
