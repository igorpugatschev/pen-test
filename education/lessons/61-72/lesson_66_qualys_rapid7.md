# Занятие 66. Qualys / Rapid7: видеообзоры, документация

## Теория

### Qualys VMDR (Vulnerability Management, Detection, and Response)

Qualys — облачная платформа для управления уязвимостями и соответствием стандартам.

**Основные модули Qualys:**
1. **VMDR** — управление уязвимостями
2. **Asset Inventory** — инвентаризация активов
3. **Threat Protection** — защита от угроз (корреляция с Threat Intel)
4. **Patch Management** — управление патчами
5. **Policy Compliance** — соответствие политикам (PCI DSS, HIPAA, ISO)
6. **File Integrity Monitoring (FIM)** — мониторинг целостности файлов
7. **Web Application Scanning (WAS)** — сканирование веб-приложений
8. **Container Security** — безопасность контейнеров

**Архитектура Qualys:**
- **Qualys Cloud Platform** — облако, где хранятся данные и происходит анализ
- **Qualys Sensors/Scanners** — виртуальные или аппаратные устройства для сканирования
- **Qualys Agents** — агенты на конечных точках (не требуют сканирования сети)
- **Qualys Web Interface** — консоль управления

**Преимущества Qualys:**
- SaaS (не нужно поддерживать сервер)
- Глобальная облачная инфраструктура
- Автоматическое обновление базы уязвимостей
- Хорошая интеграция с облачными провайдерами (AWS, Azure, GCP)
- Мощный Asset Management

**Недостатки:**
- Дорого (от $1500/год за небольшую инфраструктуру)
- Зависимость от облака (требует интернет)
- Сложная лицензионная модель

### Rapid7 InsightVM (ранее Nexpose)

Rapid7 InsightVM — платформа для управления уязвимостями с акцентом на аналитику и риски.

**Основные возможности:**
1. **Adaptive Security** — адаптивная безопасность (приоритизация рисков)
2. **Live Dashboard** — живые дашборды
3. **Remediation Projects** — проекты по устранению
4. **Metasploit Integration** — интеграция с фреймворком эксплуатации
5. **Asset Discovery** — автоматическое обнаружение активов
6. **Container Assessment** — оценка контейнеров
7. **Cloud Assessment** — оценка облачных ресурсов

**Архитектура Rapid7:**
- **InsightVM Console** — веб-интерфейс
- **Scan Engines** — движки сканирования (можно распределить)
- **Agents** — агенты для постоянного мониторинга
- **Insight Cloud** — облачная аналитика

**Преимущества Rapid7:**
- Отличная визуализация и дашборды
- Хорошая интеграция с Metasploit (подтверждение уязвимостей)
- Умная приоритизация рисков (Risk Score)
- Удобный API для автоматизации
- Хорошая поддержка Agile/DevOps процессов

**Недостатки:**
- Дорого
- Требует мощного сервера для консоли
- Сложная настройка распределенных сканирований

### Сравнение Qualys и Rapid7

| Характеристика | Qualys VMDR | Rapid7 InsightVM |
|----------------|-------------|------------------|
| Тип | SaaS (облако) | On-premise + Cloud |
| Цена | От $1500/год | От $2000/год |
| Asset Management | Отличный | Хороший |
| Визуализация | Хорошая | Отличная |
| API | Полный | Полный |
| Поддержка облаков | Нативная (AWS/Azure/GCP) | Через интеграции |
| Метасплоит интеграция | Нет | Да |
| Патч менеджмент | Да | Через сторонние инструменты |
| Сложность внедрения | Низкая (SaaS) | Средняя (требует сервер) |
