# Занятие 49. TryHackMe (Intro): путь "Complete Beginner"

## Теория

TryHackMe (THM) — это платформа для обучения кибербезопасности через практические лабораторные работы. Платформа предоставляет:

- **Виртуальные машины в браузере** (AttackBox) — готовая среда для атак
- **VPN-подключение** (OpenVPN) — для работы со своей Kali Linux
- **Треки обучения** — структурированные пути от новичка до профи
- **CTF-комнаты** — изолированные лаборатории с уязвимыми сервисами

**Формат флагов TryHackMe:** `THM{...}`

Путь **Complete Beginner** включает базовые навыки:
- Основы Linux и Windows
- Разведка (Reconnaissance)
- Сканирование портов (Nmap)
- Базовый взлом веб-приложений
- Повышение привилегий

**Как работает платформа:**
1. Регистрация на tryhackme.com
2. Подключение через VPN или использование AttackBox
3. Решение заданий в комнатах (Rooms)
4. Получение баллов и значков

**Интерфейс платформы:**
- Кнопка **"Start AttackBox"** — запускает виртуальную машину в браузере (оранжевая кнопка в правом верхнем углу комнаты)
- Кнопка **"Spawn Machine"** — запускает целевую машину для атаки (синяя кнопка в задании)
- Кнопка **"Download VPN Configuration"** — скачивание конфига для подключения (в разделе "Access")

**Адаптация под macOS (M2, 8GB):**
- Для VPN используйте **Tunnelblick** (бесплатный OpenVPN клиент для macOS) вместо командной строки: скачайте .ovpn файл и откройте через Tunnelblick
- AttackBox работает в любом браузере, виртуалка не нужна
- На 8GB RAM не запускайте локальные VM — используйте только AttackBox или очень легкие образы через UTM

## Практическое занятие

### Подготовка
1. Зарегистрируйтесь на https://tryhackme.com
2. Настройте подключение через VPN:
   - Скачайте конфиг: "Access" → "Download VPN Configuration"
   - Установите [Tunnelblick](https://tunnelblick.net/) и откройте скачанный .ovpn файл
3. Или используйте AttackBox (кнопка "Start AttackBox" в комнате)

### Прохождение комнат из трека "Complete Beginner"

#### Комната: "Intro to Research"
- Изучите поиск информации через поисковые системы
- Найдите ответы на вопросы комнаты, используя Google и документацию
- Практика: поиск CVE, чтение security advisories

#### Комната: "Linux Fundamentals 1"
- Основные команды: `ls`, `cd`, `cat`, `pwd`, `whoami`
- Работа с файлами: `touch`, `mkdir`, `rm`, `cp`, `mv`
- Практика: выполните все задания в интерактивной оболочке

#### Комната: "Nmap"
- Синтаксис: `nmap -sV -sC -p- <target>`
- Типы сканирования: TCP Connect, SYN Scan, UDP Scan
- Практика: отсканируйте целевую машину, определите открытые порты и версии сервисов

#### Комната: "Web Fundamentals"
- Структура HTTP-запросов
- Методы GET/POST
- Заголовки User-Agent, Cookie
- Практика: изучите страницу через браузер и DevTools

## Частые ошибки

1. **Проблема с VPN на macOS**: Tunnelblick иногда не подключается. Решение: скачайте свежий конфиг из раздела "Access", проверьте, что не запущен Parallels/VMware.
2. **AttackBox не загружается**: Очистите кэш браузера, попробуйте другой браузер (Chrome/Safari).
3. **Не находится флаг**: Формат флага TryHackMe: `THM{...}`. Ищите подсказки (нажмите кнопку "Hint" в комнате).
